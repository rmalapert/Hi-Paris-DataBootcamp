# Hi! PARIS Data Bootcamp

A curated collection of projects, notebooks, and exercises from the Hi! PARIS Data Boot Camp—an intensive 5-day summer course in Python, Data Science, and AI offered by Hi! PARIS (a joint initiative of HEC Paris and Institut Polytechnique de Paris, with support from Inria).

This repository includes:
- **Python Intermediate**, **Data Cleaning**, **Data Visualization**, **Machine Learning I & II**, **Miscellaneous**, and **Deliverables** notebooks (as structured in the repository).
- Hands-on exercises that take you from foundational data cleaning and visualization to building and evaluating machine learning models.
